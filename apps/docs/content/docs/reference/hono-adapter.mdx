# Hono Adapter

The `@questpie/hono` package connects the CMS engine to a web server.

## `questpieHono(cms, options)`

### Parameters
- `cms`: The `QCMS` instance.
- `options`: See configuration below.

### Configuration

---type-table---
path: ../../../packages/hono/src/server.ts
name: HonoAdapterConfig
---

### Context Middleware

Use `questpieMiddleware(cms)` to populate `c.get("cmsContext")` and `c.get("user")` for your custom routes. The CMS handler itself is a catch-all under `basePath`.

## Example

```typescript
import { Hono } from "hono";
import { questpieHono, questpieMiddleware } from "@questpie/hono";
import { cms } from "./cms";

const app = new Hono();

app.use(questpieMiddleware(cms));
app.route("/", questpieHono(cms, { basePath: "/cms" }));
```
