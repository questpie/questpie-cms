# Functions API

## `fn(options)`

```ts
import { fn } from "questpie";
import z from "zod";

export default fn({
  schema: z.object({ ... }),
  handler: async (context) => { ... },
});
```

### Options

| Option | Type | Description |
|---|---|---|
| `schema` | `ZodSchema` | Input validation schema |
| `handler` | `(ctx) => Promise<T>` | Handler function |
| `outputSchema` | `ZodSchema` | Output type validation |
| `access` | `boolean \| (ctx) => boolean` | Access control |

### Handler Context

| Property | Type | Description |
|---|---|---|
| `input` | `z.infer<schema>` | Validated input |
| `collections` | `AppCollections` | Collection API |
| `globals` | `AppGlobals` | Globals API |
| `db` | `Database` | Drizzle instance |
| `session` | `Session \| null` | Auth session |
| `queue` | `QueueClient` | Job queue |
| `email` | `MailerService` | Email service |
| `logger` | `Logger` | Logger |
| `kv` | `KVStore` | Key-value store |
| `app` | `QuestpieApp` | App instance |
| Custom services | — | Services from `services/` |

## `job(options)`

```ts
import { job } from "questpie";

export default job({
  name: "jobName",
  schema: z.object({ ... }),
  handler: async (context) => { ... },
});
```

### Job Options

| Option | Type | Description |
|---|---|---|
| `name` | `string` | Job identifier |
| `schema` | `ZodSchema` | Payload schema |
| `handler` | `(ctx) => Promise<void>` | Handler function |
| `options` | `{ retryLimit, retryDelay }` | Retry config |

### Job Context

Same as function context, but `input` is replaced by `payload`.

## `service(options)`

```ts
import { service } from "questpie";

export default service({
  lifecycle: "singleton",
  create: () => new MyService(),
});
```

| Option | Type | Description |
|---|---|---|
| `lifecycle` | `"singleton" \| "request"` | Instance lifecycle |
| `create` | `() => T` | Factory function |
| `dispose` | `(instance) => void` | Cleanup function |

## `route(options)`

```ts
import { route } from "questpie";

export default route({
  method: "GET",
  handler: async ({ request, db }) => new Response("OK"),
});
```

| Option | Type | Description |
|---|---|---|
| `method` | `"GET" \| "POST" \| "PATCH" \| "DELETE"` | HTTP method |
| `handler` | `(ctx) => Promise<Response>` | Handler |

## `email(options)`

```ts
import { email } from "questpie";

export default email({
  name: "templateName",
  schema: z.object({ ... }),
  handler: ({ input }) => ({ subject: "...", html: "..." }),
});
```

| Option | Type | Description |
|---|---|---|
| `name` | `string` | Template identifier |
| `schema` | `ZodSchema` | Input schema |
| `handler` | `(ctx) => { subject, html }` | Template renderer |
