# Field API

All fields are accessed through the `f` builder in `.fields(({ f }) => {...})`.

## Text Fields

### `f.text(options?)`

| Option | Type | Default | Description |
|---|---|---|---|
| `required` | `boolean` | `false` | Required field |
| `maxLength` | `number` | — | Max string length |
| `default` | `string` | — | Default value |
| `localized` | `boolean` | `false` | Per-locale values |
| `input` | `"optional"` | — | Optional in API input |
| `label` | `string \| i18n` | — | Display label |
| `description` | `string \| i18n` | — | Help text |

### `f.textarea(options?)`

Same options as `f.text()`. Renders as textarea in admin.

### `f.richText(options?)`

Same options as `f.text()`. Renders as rich text editor in admin.

### `f.email(options?)`

Same as `f.text()` with email format validation.

### `f.url(options?)`

Same as `f.text()` with URL format validation.

### `f.relation(options)`

| Option | Type | Default | Description |
|---|---|---|---|
| `to` | `string` | — | Target collection |

| `hasMany` | `boolean` | `false` | Has-many relation |
| `through` | `string` | — | Junction collection |

## Common Options

All fields support:

| Option | Type | Description |
|---|---|---|
| `label` | `string \| Record<string, string>` | Display label |
| `description` | `string \| Record<string, string>` | Help text |
| `required` | `boolean` | Required field |
| `default` | `T` | Default value |
| `localized` | `boolean` | Per-locale values |
| `meta` | `object` | Admin rendering hints |
| `virtual` | `SQL` | SQL computed value |
| `input` | `"optional"` | Optional in API input |
