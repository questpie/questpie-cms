# Table View

The table view displays collection items in a sortable, filterable data grid.

## Basic Configuration

```typescript
.list(({ v, f }) => v.table({
  columns: [f.title, f.status, f.author, f.createdAt],
}))
```

## Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `columns` | `ColumnConfig[]` | all fields | Columns to display |
| `defaultSort` | `SortConfig` | - | Default sort order |
| `searchable` | `boolean` | `true` | Enable search |
| `searchFields` | `string[]` | searchable fields | Fields to search |
| `filters` | `FilterConfig[]` | - | Filter controls |
| `pageSize` | `number` | `25` | Items per page |
| `pageSizeOptions` | `number[]` | `[10, 25, 50, 100]` | Page size options |
| `bulkActions` | `string[]` | `["delete"]` | Bulk operations |
| `rowAction` | `"edit" \| "view" \| "none"` | `"edit"` | Row click behavior |
| `selectable` | `boolean` | `true` | Enable row selection |

## Columns

### Simple Column

Reference a field directly:

```typescript
columns: [f.title, f.status, f.createdAt]
```

### Column Configuration

```typescript
columns: [
  f.title,
  {
    field: f.description,
    width: 200,           // Fixed width
    minWidth: 100,        // Minimum width
    maxWidth: 300,        // Maximum width
    align: "left",        // "left" | "center" | "right"
    sortable: true,       // Enable sorting
    hidden: false,        // Initially hidden
  },
  f.status,
]
```

### Custom Rendering

```typescript
columns: [
  f.title,
  {
    field: f.price,
    render: (value) => `$${(value / 100).toFixed(2)}`,
  },
  {
    field: f.status,
    render: (value, row) => (
      <Badge variant={getStatusVariant(value)}>
        {value}
      </Badge>
    ),
  },
]
```

### Computed Columns

```typescript
columns: [
  f.firstName,
  f.lastName,
  {
    key: "fullName",
    label: "Full Name",
    render: (_, row) => `${row.firstName} ${row.lastName}`,
    sortable: false,
  },
]
```

## Sorting

### Default Sort

```typescript
.list(({ v, f }) => v.table({
  columns: [f.title, f.createdAt],
  defaultSort: {
    field: "createdAt",
    direction: "desc",  // "asc" | "desc"
  },
}))
```

### Disable Sorting

```typescript
columns: [
  {
    field: f.actions,
    sortable: false,
  },
]
```

## Search

### Enable Search

```typescript
.list(({ v, f }) => v.table({
  columns: [...],
  searchable: true,
  searchFields: ["title", "description", "author.name"],
}))
```

### Disable Search

```typescript
.list(({ v, f }) => v.table({
  columns: [...],
  searchable: false,
}))
```

## Filters

### Field Filters

```typescript
.list(({ v, f }) => v.table({
  columns: [...],
  filters: [
    f.status,      // Auto-generates filter from field type
    f.author,      // Relation filter
    f.category,    // Select filter
  ],
}))
```

### Custom Filter Configuration

```typescript
filters: [
  {
    field: f.status,
    label: "Filter by Status",
    defaultValue: "published",
  },
  {
    field: f.createdAt,
    label: "Date Range",
    type: "dateRange",
  },
]
```

### Filter Types

| Field Type | Filter Type |
|------------|-------------|
| `select` | Dropdown |
| `checkbox/switch` | Yes/No toggle |
| `relation` | Entity picker |
| `date/datetime` | Date range |
| `number` | Range slider |

## Pagination

```typescript
.list(({ v, f }) => v.table({
  columns: [...],
  pageSize: 50,
  pageSizeOptions: [25, 50, 100, 200],
}))
```

## Bulk Actions

```typescript
.list(({ v, f }) => v.table({
  columns: [...],
  bulkActions: [
    "delete",
    {
      id: "publish",
      label: "Publish Selected",
      icon: CheckIcon,
      action: async (ids) => {
        await client.collections.posts.updateMany({
          where: { id: { in: ids } },
          data: { status: "published" },
        });
      },
    },
    {
      id: "archive",
      label: "Archive Selected",
      icon: ArchiveIcon,
      variant: "warning",
      action: async (ids) => {...},
    },
  ],
}))
```

## Row Actions

### Click Action

```typescript
.list(({ v, f }) => v.table({
  columns: [...],
  rowAction: "edit",    // Click row to edit
  // rowAction: "view",  // Click row to view
  // rowAction: "none",  // No click action
}))
```

### Action Column

```typescript
columns: [
  f.title,
  f.status,
  {
    key: "actions",
    label: "",
    width: 100,
    render: (_, row) => (
      <DropdownMenu>
        <DropdownMenuItem onClick={() => edit(row.id)}>
          Edit
        </DropdownMenuItem>
        <DropdownMenuItem onClick={() => duplicate(row.id)}>
          Duplicate
        </DropdownMenuItem>
        <DropdownMenuItem onClick={() => delete(row.id)}>
          Delete
        </DropdownMenuItem>
      </DropdownMenu>
    ),
  },
]
```

## Empty State

```typescript
.list(({ v, f }) => v.table({
  columns: [...],
  emptyState: {
    title: "No posts yet",
    description: "Create your first blog post",
    action: {
      label: "Create Post",
      href: "/admin/collections/posts/create",
    },
  },
}))
```

## Complete Example

```typescript
export const appointmentsAdmin = builder
  .collection("appointments")
  .meta({
    label: "Appointments",
    icon: CalendarIcon,
  })
  .fields(({ r }) => ({
    customerName: r.text({ label: "Customer" }),
    customerEmail: r.email({ label: "Email" }),
    barber: r.relation({ label: "Barber", targetCollection: "barbers" }),
    service: r.relation({ label: "Service", targetCollection: "services" }),
    scheduledAt: r.datetime({ label: "Date & Time" }),
    status: r.select({
      label: "Status",
      options: [
        { label: "Pending", value: "pending", color: "yellow" },
        { label: "Confirmed", value: "confirmed", color: "blue" },
        { label: "Completed", value: "completed", color: "green" },
        { label: "Cancelled", value: "cancelled", color: "red" },
      ],
    }),
  }))
  .list(({ v, f }) => v.table({
    columns: [
      f.customerName,
      f.barber,
      f.service,
      f.scheduledAt,
      {
        field: f.status,
        width: 120,
      },
    ],
    defaultSort: {
      field: "scheduledAt",
      direction: "desc",
    },
    searchable: true,
    searchFields: ["customerName", "customerEmail"],
    filters: [f.status, f.barber, f.service],
    pageSize: 25,
    bulkActions: [
      "delete",
      {
        id: "confirm",
        label: "Confirm Selected",
        icon: CheckIcon,
        action: async (ids) => {
          // Bulk confirm logic
        },
      },
    ],
    emptyState: {
      title: "No appointments",
      description: "No appointments scheduled yet",
    },
  }));
```
