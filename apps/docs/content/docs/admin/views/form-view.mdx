# Form View

The form view configures how items are created and edited in the admin interface.

## Basic Configuration

```typescript
.form(({ v, f }) => v.form({
  fields: [f.title, f.content, f.status],
}))
```

## Options

| Option | Type | Description |
|--------|------|-------------|
| `fields` | `FieldConfig[]` | Simple field list |
| `sections` | `SectionConfig[]` | Organized sections |
| `tabs` | `TabConfig[]` | Tabbed interface |
| `sidebar` | `SidebarConfig` | Sidebar with fields |
| `layout` | `"default" \| "with-sidebar"` | Layout type |

## Field List

Simplest form - just list fields:

```typescript
.form(({ v, f }) => v.form({
  fields: [f.title, f.slug, f.content, f.status],
}))
```

## Sections

Organize fields into logical groups:

```typescript
.form(({ v, f }) => v.form({
  sections: [
    {
      title: "Content",
      description: "Main article content",
      fields: [f.title, f.slug, f.content],
    },
    {
      title: "Publishing",
      fields: [f.status, f.publishedAt, f.author],
    },
    {
      title: "SEO",
      collapsible: true,
      defaultOpen: false,
      fields: [f.metaTitle, f.metaDescription],
    },
  ],
}))
```

### Section Options

| Option | Type | Description |
|--------|------|-------------|
| `title` | `string \| I18nText` | Section header |
| `description` | `string \| I18nText` | Help text |
| `fields` | `FieldConfig[]` | Fields in section |
| `collapsible` | `boolean` | Can be collapsed |
| `defaultOpen` | `boolean` | Initial state |
| `layout` | `"stack" \| "columns" \| "grid"` | Field arrangement |
| `columns` | `number` | Column count (for columns layout) |
| `grid` | `GridConfig` | Grid configuration |

## Sidebar

Add a sidebar for metadata fields:

```typescript
.form(({ v, f }) => v.form({
  layout: "with-sidebar",
  sidebar: {
    position: "right",    // "left" | "right"
    width: "300px",       // CSS width
    fields: [f.status, f.publishedAt, f.author, f.featuredImage],
  },
  sections: [
    {
      title: "Content",
      fields: [f.title, f.content],
    },
  ],
}))
```

### Sidebar Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `position` | `"left" \| "right"` | `"right"` | Sidebar position |
| `width` | `string` | `"300px"` | Sidebar width |
| `fields` | `FieldConfig[]` | - | Fields in sidebar |
| `sticky` | `boolean` | `true` | Sticky positioning |

## Tabs

Organize content in tabs:

```typescript
.form(({ v, f }) => v.form({
  tabs: [
    {
      id: "content",
      label: "Content",
      icon: FileTextIcon,
      fields: [f.title, f.content],
    },
    {
      id: "media",
      label: "Media",
      icon: ImageIcon,
      fields: [f.featuredImage, f.gallery],
    },
    {
      id: "seo",
      label: "SEO",
      icon: SearchIcon,
      fields: [f.metaTitle, f.metaDescription, f.canonicalUrl],
    },
  ],
}))
```

### Tab Options

| Option | Type | Description |
|--------|------|-------------|
| `id` | `string` | Unique identifier |
| `label` | `string \| I18nText` | Tab label |
| `icon` | `IconComponent` | Tab icon |
| `fields` | `FieldConfig[]` | Tab content |

## Tabs with Sidebar

Combine tabs and sidebar:

```typescript
.form(({ v, f }) => v.form({
  sidebar: {
    position: "right",
    fields: [f.status, f.publishedAt, f.author],
  },
  tabs: [
    {
      id: "content",
      label: "Content",
      fields: [f.title, f.content],
    },
    {
      id: "seo",
      label: "SEO",
      fields: [f.metaTitle, f.metaDescription],
    },
  ],
}))
```

## Field Configuration

### Simple Field Reference

```typescript
fields: [f.title, f.content]
```

### Field with Span (Grid Layout)

```typescript
{
  layout: "grid",
  grid: { columns: 4 },
  fields: [
    { field: f.title, span: 4 },      // Full width
    { field: f.firstName, span: 2 },  // Half width
    { field: f.lastName, span: 2 },   // Half width
  ],
}
```

### Nested Sections

```typescript
fields: [
  f.title,
  {
    type: "section",
    title: "Contact Info",
    layout: "columns",
    columns: 2,
    fields: [f.email, f.phone],
  },
  f.content,
]
```

## Complete Example

```typescript
export const barbersAdmin = builder
  .collection("barbers")
  .meta({
    label: { en: "Barbers", sk: "Holici" },
    icon: UsersIcon,
  })
  .fields(({ r }) => ({
    name: r.text({ label: "Name", required: true }),
    email: r.email({ label: "Email" }),
    phone: r.text({ label: "Phone" }),
    bio: r.richText({ label: "Biography" }),
    avatar: r.upload({ label: "Photo", accept: ["image/*"] }),
    isActive: r.switch({ label: "Active" }),
    services: r.relation({
      label: "Services",
      targetCollection: "services",
      type: "multiple",
    }),
    workingHours: r.object({
      label: "Working Hours",
      fields: ({ r }) => ({
        monday: r.object({...}),
        // ...
      }),
    }),
    socialLinks: r.array({
      label: "Social Links",
      item: ({ r }) => ({...}),
    }),
  }))
  .form(({ v, f }) => v.form({
    sidebar: {
      position: "right",
      fields: [f.isActive, f.avatar],
    },
    sections: [
      {
        title: { en: "Contact", sk: "Kontakt" },
        layout: "grid",
        grid: { columns: 2 },
        fields: [f.name, f.email, f.phone],
      },
      {
        title: { en: "Profile", sk: "Profil" },
        fields: [f.bio],
      },
      {
        title: { en: "Services", sk: "Sluzby" },
        fields: [f.services],
      },
      {
        title: { en: "Social", sk: "Socialne siete" },
        collapsible: true,
        fields: [f.socialLinks],
      },
      {
        title: { en: "Schedule", sk: "Rozvrh" },
        collapsible: true,
        fields: [f.workingHours],
      },
    ],
  }));
```

## Form Actions

The form automatically includes:

- **Save** - Save and stay on form
- **Save & Close** - Save and return to list
- **Cancel** - Discard changes
- **Delete** - Delete item (with confirmation)

Custom actions can be added via hooks or custom components.
