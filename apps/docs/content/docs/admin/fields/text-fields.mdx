# Text Fields

Text fields handle string input in various formats.

## text

Single-line text input for short strings.

```typescript
title: r.text({
  label: "Title",
  placeholder: "Enter title...",
  maxLength: 200,
  minLength: 3,
  pattern: "^[a-z-]+$",  // Regex validation
  required: true,
})
```

### Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `label` | `string \| I18nText` | - | Display label |
| `placeholder` | `string \| I18nText` | - | Placeholder text |
| `description` | `string \| I18nText` | - | Help text |
| `maxLength` | `number` | - | Maximum characters |
| `minLength` | `number` | - | Minimum characters |
| `pattern` | `string` | - | Regex pattern for validation |
| `required` | `boolean \| fn` | `false` | Required validation |
| `readOnly` | `boolean \| fn` | `false` | Disable editing |
| `visible` | `fn` | - | Conditional visibility |
| `defaultValue` | `string` | - | Default value |
| `localized` | `boolean` | `false` | Store per-locale values |

### Examples

```typescript
// Basic title field
title: r.text({
  label: "Title",
  maxLength: 200,
  required: true,
})

// Slug field with pattern
slug: r.text({
  label: "URL Slug",
  placeholder: "my-post-title",
  pattern: "^[a-z0-9-]+$",
  description: "Lowercase letters, numbers, and hyphens only",
})

// Phone number
phone: r.text({
  label: "Phone",
  placeholder: "+1 (555) 000-0000",
})

// Localized title
title: r.text({
  label: { en: "Title", sk: "Titulok" },
  localized: true,  // Separate value per locale
})
```

## email

Email input with built-in validation.

```typescript
email: r.email({
  label: "Email Address",
  placeholder: "user@example.com",
  required: true,
})
```

### Options

Same as `text`, plus automatic email format validation.

### Examples

```typescript
// Contact email
contactEmail: r.email({
  label: "Contact Email",
  required: true,
})

// Optional notification email
notificationEmail: r.email({
  label: "Notification Email",
  description: "Receive updates at this address",
})
```

## password

Masked password input.

```typescript
password: r.password({
  label: "Password",
  minLength: 8,
  required: true,
})
```

### Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `label` | `string \| I18nText` | - | Display label |
| `placeholder` | `string \| I18nText` | - | Placeholder text |
| `minLength` | `number` | - | Minimum length |
| `maxLength` | `number` | - | Maximum length |
| `showStrength` | `boolean` | `false` | Show strength indicator |

### Examples

```typescript
// Basic password
password: r.password({
  label: "Password",
  minLength: 8,
})

// With strength indicator
password: r.password({
  label: "Password",
  minLength: 8,
  showStrength: true,
  description: "At least 8 characters with uppercase, lowercase, and numbers",
})
```

## textarea

Multi-line text input for longer content.

```typescript
description: r.textarea({
  label: "Description",
  rows: 5,
  maxLength: 1000,
})
```

### Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `label` | `string \| I18nText` | - | Display label |
| `placeholder` | `string \| I18nText` | - | Placeholder text |
| `rows` | `number` | `3` | Visible rows |
| `minRows` | `number` | - | Minimum rows (auto-resize) |
| `maxRows` | `number` | - | Maximum rows (auto-resize) |
| `maxLength` | `number` | - | Maximum characters |
| `minLength` | `number` | - | Minimum characters |
| `resize` | `"none" \| "vertical" \| "auto"` | `"vertical"` | Resize behavior |

### Examples

```typescript
// Fixed height
description: r.textarea({
  label: "Description",
  rows: 5,
  maxLength: 500,
})

// Auto-resizing
bio: r.textarea({
  label: "Biography",
  minRows: 3,
  maxRows: 10,
  resize: "auto",
})

// Non-resizable
notes: r.textarea({
  label: "Internal Notes",
  rows: 4,
  resize: "none",
})

// Localized content
description: r.textarea({
  label: { en: "Description", sk: "Popis" },
  localized: true,
  rows: 5,
})
```

## Conditional Visibility

All text fields support conditional visibility:

```typescript
status: r.select({
  label: "Status",
  options: [
    { label: "Active", value: "active" },
    { label: "Cancelled", value: "cancelled" },
  ],
}),

// Show only when cancelled
cancellationReason: r.textarea({
  label: "Cancellation Reason",
  visible: (values) => values.status === "cancelled",
  required: (values) => values.status === "cancelled",
}),
```

## Table Display

Text fields display their value directly in tables. Long text is truncated with ellipsis.

For custom table display, you can configure the cell:

```typescript
.list(({ v, f }) => v.table({
  columns: [
    f.title,
    {
      field: f.description,
      maxWidth: 200,  // Truncate long descriptions
    },
  ],
}))
```
