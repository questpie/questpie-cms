# Installation

QUESTPIE CMS is distributed as a set of NPM packages. It requires a Node.js or Bun environment and a PostgreSQL database.

## Prerequisites

- **Runtime**: Node.js 18+ or Bun 1.0+
- **Database**: PostgreSQL 14+
- **Language**: TypeScript 5.0+

## Package Installation

Install the core package and the necessary database drivers.

<Tabs items={['bun', 'npm', 'pnpm', 'yarn']}>
  <Tab value="bun">
    ```bash
    bun add @qcms/core drizzle-orm postgres
    bun add -d drizzle-kit
    ```
  </Tab>
  <Tab value="npm">
    ```bash
    npm install @qcms/core drizzle-orm postgres
    npm install -D drizzle-kit
    ```
  </Tab>
  <Tab value="pnpm">
    ```bash
    pnpm add @qcms/core drizzle-orm postgres
    pnpm add -D drizzle-kit
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn add @qcms/core drizzle-orm postgres
    yarn add -D drizzle-kit
    ```
  </Tab>
</Tabs>

## Database Configuration

QUESTPIE uses Drizzle ORM for database connectivity. Initialize your database client and export the Drizzle instance.

```typescript title="db/index.ts"
import { drizzle } from 'drizzle-orm/postgres-js'
import postgres from 'postgres'

const client = postgres(process.env.DATABASE_URL!)
export const db = drizzle(client)
```

## Environment Variables

Configure your database connection string in a `.env` file.

```bash title=".env"
DATABASE_URL=postgresql://user:password@localhost:5432/mydb
```

## TypeScript Configuration

QUESTPIE relies on advanced TypeScript features. Ensure `strict` mode is enabled in your `tsconfig.json`.

```json title="tsconfig.json"
{
  "compilerOptions": {
    "strict": true,
    "target": "ES2022",
    "module": "ESNext",
    "moduleResolution": "bundler",
    "skipLibCheck": true
  }
}
```
