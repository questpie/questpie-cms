# Globals

A global is a **singleton** — a single record with no list view, just a form. Use globals for site-wide configuration: settings, SEO defaults, navigation, footer content.

## Defining a Global

```ts title="globals/site-settings.ts"
import { global } from "#questpie";

export const siteSettings = global("site_settings")
  .fields(({ f }) => ({
    shopName: f.text({ required: true, default: "My App" }),
    tagline: f.text({ localized: true }),
    logo: f.upload({ to: "assets" }),
    contactEmail: f.email({ required: true }),
  }))
  .admin(({ c }) => ({
    label: { en: "Site Settings", sk: "Nastavenia webu" },
    icon: c.icon("ph:gear"),
  }))
  .options({
    timestamps: true,
    versioning: true,
  })
  .access({
    read: true,
    update: ({ session }) => (session?.user as any)?.role === "admin",
  });
```

## Builder Chain

Globals share most methods with collections:

| Method | Purpose |
|---|---|
| `.fields(({ f }) => {...})` | Define data fields |
| `.admin(({ c }) => {...})` | Admin label and icon |
| `.form(({ v, f }) => v.form({...}))` | Form layout |
| `.hooks({...})` | Lifecycle hooks |
| `.access({...})` | Read/update access control |
| `.options({...})` | Timestamps, versioning |

Globals do **not** support `.list()`, `.indexes()`, `.title()`, or `.preview()` since there's only ever one record.

## Server-Side API

```ts
// Read
const settings = await globals.siteSettings.get();

// Update
await globals.siteSettings.update({
  shopName: "New Name",
  tagline: "New tagline",
});
```

## Client-Side API

```ts
// Read
const settings = await client.globals.siteSettings.get();

// Update (requires access)
await client.globals.siteSettings.update({
  shopName: "New Name",
});
```

## Real-World Example

From the barbershop — site settings with nested objects, arrays, and access control:

```ts title="globals/site-settings.ts"
import { global } from "#questpie";

export const siteSettings = global("site_settings")
  .fields(({ f }) => ({
    shopName: f.text({ required: true, default: "Sharp Cuts" }),
    tagline: f.text({ localized: true }),
    logo: f.upload({ to: "assets" }),

    navigation: f.array({
      localized: true,
      of: f.object({
        fields: {
          label: f.text({ required: true }),
          href: f.text({ required: true }),
          isExternal: f.boolean({ default: false }),
        },
      }),
    }),

    contactEmail: f.email({ required: true }),
    contactPhone: f.text(),
    address: f.text(),

    businessHours: f.object({
      fields: {
        monday: f.object({
          fields: {
            isOpen: f.boolean({ default: true }),
            start: f.time(),
            end: f.time(),
          },
        }),
        // ... other days
      },
    }),

    bookingSettings: f.object({
      fields: {
        minAdvanceHours: f.number({ required: true }),
        maxAdvanceDays: f.number({ required: true }),
        slotDurationMinutes: f.number({ required: true }),
        allowCancellation: f.boolean({ required: true }),
      },
    }),

    metaTitle: f.text({ localized: true }),
    metaDescription: f.textarea({ localized: true }),
  }))
  .admin(({ c }) => ({
    label: { en: "Site Settings", sk: "Nastavenia webu" },
    icon: c.icon("ph:gear"),
  }))
  .form(({ v, f }) =>
    v.form({
      fields: [
        {
          type: "section",
          label: { en: "Branding" },
          layout: "grid",
          columns: 2,
          fields: [f.shopName, f.tagline, f.logo],
        },
        {
          type: "section",
          label: { en: "Contact" },
          layout: "grid",
          columns: 2,
          fields: [f.contactEmail, f.contactPhone, f.address],
        },
        // ... more sections
      ],
    }),
  )
  .options({ timestamps: true, versioning: true })
  .access({
    read: true,
    update: ({ session }) => (session?.user as any)?.role === "admin",
  });
```
