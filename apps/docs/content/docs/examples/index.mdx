# Examples

These examples demonstrate real-world QuestPie patterns using actual code from the repository.

## [TanStack Barbershop](/docs/examples/barbershop)

A full booking application built with TanStack Start + QuestPie. Features:

- 7 collections with complex relations
- Server functions for booking logic
- Background jobs for email notifications
- Custom services (blog utilities)
- Admin dashboard with stats, charts, and timelines
- Block-based page builder
- Multi-language support (English + Slovak)
- Live preview
