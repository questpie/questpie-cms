# TanStack Barbershop Architecture Tour

This example demonstrates the v2 architecture in a real app.

## Core structure

- server builder and CMS composition
- server-defined collections/globals/dashboard/sidebar
- typed rpc functions for booking and stats
- client admin builder for rendering and locale

## File map

- `examples/tanstack-barbershop/src/questpie/server/builder.ts`
- `examples/tanstack-barbershop/src/questpie/server/cms.ts`
- `examples/tanstack-barbershop/src/questpie/server/collections/*.ts`
- `examples/tanstack-barbershop/src/questpie/server/functions/*.ts`
- `examples/tanstack-barbershop/src/questpie/admin/builder.ts`
- `examples/tanstack-barbershop/src/questpie/admin/admin.ts`

## Why this example matters

It shows the core pattern you should copy:

- `questpie` core for system contracts
- `@questpie/admin/server` to extend builder with admin capabilities
- `@questpie/admin/client` to resolve rendering via registries
