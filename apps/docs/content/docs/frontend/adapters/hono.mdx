# Hono Adapter

The Hono adapter connects QuestPie to the [Hono](https://hono.dev) web framework.

## Install

```bash
bun add @questpie/hono hono
```

## Setup

```ts title="src/index.ts"
import { Hono } from "hono";
import { createFetchHandler } from "questpie/adapters/hono";
import { app } from "@/questpie/server/.generated";

const server = new Hono();

const handler = createFetchHandler(app, { basePath: "/api" });

server.all("/api/*", (c) => handler(c.req.raw));

export default server;
```

## With OpenAPI

```ts
import { withOpenApi } from "questpie/openapi";

const handler = withOpenApi(
  createFetchHandler(app, { basePath: "/api" }),
  { basePath: "/api" },
);

// OpenAPI spec at /api/openapi.json
// Scalar UI at /api/docs
```
